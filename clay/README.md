# Clay BuildSystem

## Overview

Built using Golang, Clay is a build system designed to be simple, fast, and efficient. 

## Features

- Platforms:
  - linux
  - macOS (darwin)
  - windows
- Compilers:
  - Clang (darwin and linux)
  - MSVC (windows)
- Custom Dependency Tracker
- Targets
  - darwin
    * darwin
    * arduino Esp32
  - linux
    * linux
  - windows
    * windows
