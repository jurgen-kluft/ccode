# Clay BuildSystem

## Overview

Built using Golang, Clay is a build system designed to be simple, and efficient. 

## Features

- Platforms:
  - MacOS (darwin)
    - Arduino (Both ESP32 & ESP8266)
  - Windows (wip)
  - Linux (should be simple to add)
- Compilers:
  - Clang (Darwin and Linux)
  - MSVC (Windows)
- Dependency Tracker
