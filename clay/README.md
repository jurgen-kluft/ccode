# Clay BuildSystem

## Overview

Built using Golang, Clay is a build system designed to be simple, and efficient. 

## Features

- Platforms:
  - Mac
    - MacOS
    - Arduino (Both ESP32 & ESP8266)
  - Windows (Alpha)
  - Linux 
    - should be simple to add, nearly identical to MacOS
- Compilers:
  - Clang (Darwin)
  - MSVC (Windows)
- A solid Dependency Tracker
