# Clay BuildSystem

## Overview

Built using Golang, Clay is a build system designed to be simple, and efficient. 

## Features

- Platforms:
  - macOS (darwin)
  - windows (wip)
- Compilers:
  - GCC/Clang (darwin and linux)
  - MSVC (windows)
- Dependency Tracker
- Targets
  - darwin
    * darwin
    * arduino Esp32
  - windows
    * windows
